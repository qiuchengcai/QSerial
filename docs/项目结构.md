# QSerial 项目结构

> 版本: 2.0.0
> 更新日期: 2026-04-20

---

## 项目概述

QSerial 是一款现代化的跨平台终端工具，支持串口、SSH、Telnet、本地终端等多种连接方式，并提供连接共享、SFTP 文件传输、TFTP 服务器、快捷按钮等扩展功能。

## 核心模块说明

### 连接管理 (connection/)

| 文件 | 说明 |
|------|------|
| `factory.ts` | 连接工厂，管理所有连接实例的创建/获取/销毁 |
| `pty.ts` | PTY 本地终端连接，使用 node-pty |
| `serial.ts` | 串口连接，使用 serialport，支持自动重连和连接复用 |
| `ssh.ts` | SSH 连接，使用 ssh2，兼容旧设备算法，支持密钥/密码认证 |
| `telnet.ts` | Telnet 连接，使用 net |
| `serialServer.ts` | 串口共享服务端（**已废弃**，由 connectionServer 替代） |
| `connectionServer.ts` | 连接共享服务端：TCP Server + TELNET 协议协商 + 密码认证 + 写入队列 + SSH 反向隧道，支持共享任意类型活跃连接 |

### 连接共享服务 (connectionServer.ts)

- **TCP 服务器**: 将任意活跃连接通过 TCP 共享给网络客户端
- **TELNET 协议协商**: 主动协商 WILL ECHO + SUPPRESS-GA + DO NAWS + DO TTYPE，确保远程客户端 Tab 补全、方向键等交互功能正常
- **可配置监听地址**: `0.0.0.0` (局域网可访问) 或 `127.0.0.1` (仅本机)
- **密码认证**: 可选的密码认证协议，10 秒超时，最多 3 次尝试
- **写入队列**: 串行化多客户端写入，防止数据帧交错
- **SSH 反向隧道**: 可选，自动尝试 `~/.ssh` 默认密钥，断线自动重连（指数退避，最多 5 次）
- **数据源**: 支持复用已有活跃连接（`sourceType: 'existing'`）或创建新连接（`sourceType: 'new'`）

### SFTP 文件传输 (sftp/)

| 文件 | 说明 |
|------|------|
| `manager.ts` | SFTP 会话管理器，提供目录列表/上传/下载/创建目录/删除/重命名/符号链接等操作 |

### TFTP 服务器 (tftp/)

| 文件 | 说明 |
|------|------|
| `manager.ts` | TFTP 服务器管理器，启动/停止/状态查询/传输事件监控 |
