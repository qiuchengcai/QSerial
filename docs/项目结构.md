# QSerial 项目结构

> 版本: 1.0.0
> 更新日期: 2026-04-17

---

## 项目概述

QSerial 是一款现代化的跨平台终端工具，支持串口、SSH、本地终端等多种连接方式，并提供串口共享、SFTP 文件传输、TFTP 服务器等扩展功能。

## 核心模块说明

### 连接管理 (connection/)

| 文件 | 说明 |
|------|------|
| `factory.ts` | 连接工厂，管理所有连接实例的创建/获取/销毁 |
| `pty.ts` | PTY 本地终端连接，使用 node-pty |
| `serial.ts` | 串口连接，使用 serialport，支持复用现有连接 |
| `serialServer.ts` | 串口共享服务端：TCP Server + 密码认证 + 写入队列 + SSH 反向隧道 |
| `ssh.ts` | SSH 连接，使用 ssh2 |
| `telnet.ts` | Telnet 连接 |

### 串口共享服务 (serialServer.ts)

- **TCP 服务器**: 将串口通过 TCP 共享给网络客户端
- **可配置监听地址**: `0.0.0.0` (局域网可访问) 或 `127.0.0.1` (仅本机)
- **密码认证**: 可选的 `PASSWORD:xxx` 认证协议，10 秒超时
- **写入队列**: 串行化多客户端写入，防止数据帧交错
- **SSH 反向隧道**: 可选，自动尝试 `~/.ssh` 默认密钥，断线自动重连
- **连接复用**: 自动检测并复用现有串口连接

### SFTP 文件传输 (sftp/)

| 文件 | 说明 |
|------|------|
| `manager.ts` | SFTP 会话管理器，提供文件列表/上传/下载/创建目录等操作 |

### TFTP 服务器 (tftp/)

| 文件 | 说明 |
|------|------|
| `manager.ts` | TFTP 服务器管理器，启动/停止/状态查询 |
