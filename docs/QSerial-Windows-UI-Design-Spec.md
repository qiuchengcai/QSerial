## 2. 顶部菜单栏

菜单栏采用 Windows 标准 Mnemonic 下划线风格，紧贴标题栏下方。每个菜单项的首字母带下划线，`Alt+字母` 快速展开。

| 新建串口连接(&S) | `Ctrl+Shift+N` | 打开串口连接对话框 |
| 新建 SSH 连接(&H) | `Ctrl+Shift+S` | 打开 SSH 连接对话框 |
| 新建 Telnet 连接(&T) | — | 打开 Telnet 连接对话框，默认端口 23 |
| ────────── | | |

### 2.2 菜单状态联动

- "开始日志记录"→ 仅在当前活动会话存在且未在记录日志时可用。
- "停止日志记录"→ 仅在当前活动会话正在记录日志时可用。
- "SFTP 文件浏览器"→ 仅在当前活动会话类型为 SSH 且状态为已连接时可用。
- "断开当前连接"→ 仅在当前活动会话处于已连接/连接中状态时可用。
- "重连"→ 仅在当前活动会话处于已断开/错误状态时可用。

## 3. 工具栏

### 3.1 布局

工具栏位于侧边栏和主内容区之间（或可选内嵌于标签栏右上角），作为最常用操作的快速入口。默认显示，可通过"视图"菜单隐藏。

```
┌──────┬──────┬──────┬──────┬──┬──────┬──────┬──────┬──┬──────┬──────┐
│  🔌   │  🌐   │  💻   │  📡   │  │  📁   │  📤   │  🤖   │  │  ⚙    │  ❓   │
│ 串口   │ SSH   │ 本地   │Telnet │  │ SFTP  │ 共享   │ MCP   │  │ 设置   │ 帮助  │
└──────┴──────┴──────┴──────┴──┴──────┴──────┴──────┴──┴──────┴──────┘
```

### 3.2 按钮定义

| 图标 | 中文标签 | ToolTip | 关联菜单项 |
|------|----------|---------|-----------|
| 🔌 | 串口 | 新建串口连接 (Ctrl+Shift+N) | 文件→新建串口连接 |
| 🌐 | SSH | 新建 SSH 连接 (Ctrl+Shift+S) | 文件→新建 SSH 连接 |
| 💻 | 本地终端 | 新建本地终端 (Ctrl+N) | 文件→新建本地终端 |
| 📡 | Telnet | 新建 Telnet 连接 | 文件→新建 Telnet 连接 |
| —(分组符)— | | | |
| 📁 | SFTP | 打开 SFTP 文件浏览器（仅 SSH 已连接可用） | 连接→SFTP 文件浏览器 |
| 📤 | 共享 | 打开连接共享面板 | 连接→连接共享 |
| 🤖 | MCP | 打开 MCP AI 服务器面板 | 服务→MCP AI 服务器 |
| —(分组符)— | | | |
| ⚙ | 设置 | 打开设置对话框 (Ctrl+,) | 设置→打开设置 |
| ❓ | 帮助 | QSerial 帮助 (F1) | 帮助→QSerial 帮助 |

### 3.3 显示/隐藏规则

- 工具栏可通过"视图"菜单项 "显示/隐藏工具栏" 切换。
- 当所有标签关闭时（欢迎页），连接类按钮（串口/SSH/本地/Telnet）保持可用，服务类按钮（SFTP/共享/MCP）保持可用。
- SFTP 按钮在无 SSH 已连接会话时置灰 (disabled)。

### 4.3 已保存配置的开关行为

| 当前状态 | 点击行为 | 连接后状态 |
|----------|----------|-----------|
| ○ 未连接（空心圆，灰色） | 发起连接，创建新标签 → 加载中 | ● 已连接（实心绿圆+勾） |
| ● 已连接（实心绿圆+勾） | 弹出确认对话框："断开 [名称]？" | ○ 未连接 |
| ◉ 连接中（动画旋转环，蓝色） | 不响应点击（正在连接中，禁用） | — |

> 查找重复逻辑：同名串口 path 或同 host 的 SSH/Telnet 视为同一连接，点击已连接配置执行断开而非再连一次。

### 4.4 连接状态颜色定义

| 状态 | 图标 | 颜色 | 文案 |
|------|------|------|------|
| 已断开 | ○ | `#8B9298` (次要文字色) | 点击连接 |
| 连接中 | ◉ (旋转动画) | `#3B82F6` (蓝色) | 连接中... |
| 已连接 | ● ✓ | `#22C55E` (成功绿) | 已连接·点击断开 |
| 断开中 | ◉ (闪烁) | `#F59E0B` (警告黄) | 断开中... |
| 重连中 | ◉ (旋转动画) | `#F59E0B` (警告黄) | 重连中 (N/5)... |
| 错误 | ✕ | `#EF4444` (错误红) | 连接失败·点击重试 |

### 4.5 空状态文案

- "暂无保存的配置 — 新建连接后勾选「保存为配置」即可在此处快速复用"
- （侧边栏折叠态）图标 ToolTip 显示完整连接类型名

### 4.6 键盘可达性

- `Tab` 在按钮间切换，`Enter` 激活
- 对话框内 `Enter` 提交连接，`Esc` 关闭对话框

## 5. 多标签管理区域

### 5.1 标签栏布局

```
┌──────────────────────────────────────────────────────────────────────┐
│ [● 串口 COM3  ×] [○ SSH root@192.168 ×] [○ 本地终端 ×]  [▾⋮]       │
└──────────────────────────────────────────────────────────────────────┘
```

### 5.2 标签交互

| 交互 | 行为 |
|------|------|
| 左键单击 | 切换到该标签 |
| 滚轮 | 自动转为横向滚动（`e.deltaY * 2`） |
| 拖拽 | 鼠标按住可横向拖拽排序（drag & drop） |
| 双击空白区 | 新建本地终端标签 |
| 中键单击 | 关闭该标签 |
| 关闭按钮 × | 仅在鼠标悬停或标签为当前激活时显示 |
| 右侧 ⋮ 按钮 | 下拉列出所有标签名（当前激活打 ✓），点击切换 |

### 5.3 右键菜单

```
┌─────────────────────┐
│ ✕ 关闭              │  ← 关闭当前标签
│ ✕ 关闭其他标签页    │  ← 保留当前，关闭其余
│ ✕ 关闭左侧标签页    │  ← 关闭所有靠左的
│ ✕ 关闭右侧标签页    │  ← 关闭所有靠右的
│ ──────────────      │
```

## 附录 A：全局交互规范

### A.1 Toast 错误提示

- 位置：应用右上角
- 样式：红色左边框 + 错误图标 + 错误文本
- 持续时间：5 秒自动消失，hover 时暂停计时
- 支持手动关闭按钮

### A.2 右键菜单

- 统一使用 ContextMenu 组件（portal 到 body）
- 自动检测屏幕边缘，超出时反向展开
- 点击菜单外区域自动关闭

### A.3 对话框

- 所有模态对话框使用固定定位 + 半透明黑色遮罩
- 关闭方式：右上角 × 按钮 / Esc 键 / 点击遮罩
- 统一圆角（跟随主题）+ 白色 5% 透明度边框

### A.4 键盘可达性

- Tab 键在可聚焦元素间移动
- Enter 激活当前聚焦的按钮
- Esc 关闭当前对话框/下拉菜单/右键菜单
- F1 打开帮助

### A.5 窗口拖拽与调整

- 拖拽标题栏（不含按钮区域）移动窗口
- 双击标题栏空白区域 = 最大化/还原
- 窗口边缘可拖拽调整窗口大小（系统行为）
- 侧边栏右侧边缘可拖拽调整侧边栏宽度（160–400px）

### A.6 连接失败处理

- 创建连接失败时：Toast 提示错误信息 + 自动清理已创建的标签和会话
- 自动重连失败达到上限时：状态栏显示 `✕错误` + 终端内写入 `--- 重连失败，已达最大重试次数 ---`
