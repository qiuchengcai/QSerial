# QSerial 项目文件说明

## 项目概述

QSerial 是一款现代化的跨平台终端工具，支持串口、SSH、本地终端等多种连接方式。

---

## 常用命令

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建项目
pnpm build

# 打包应用
pnpm package:win      # Windows
pnpm package:linux    # Linux
pnpm package:mac      # macOS

# 代码检查
pnpm lint
pnpm lint:fix

# 测试
pnpm test
```
