核心特性

嵌入式工程师所需的一切

从串口调试到 AI 驱动的自动化，QSerial 覆盖嵌入式开发的完整工作流。

🤖

MCP AI 集成

42 个内置 MCP 工具，让 AI 助手直接操控你的设备。Resources、Notifications、Sampling、Prompts 全部标准实现。

🔌

多协议统一

Serial (COM/tty)、SSH（支持 Jump Host 跳板）、Telnet、本地 PTY。统一界面，一致的操作体验。

🔍

自动设备识别

conn.analyze.probe 自动探测 21 种设备：ESP32、STM32、NXP i.MX、树莓派、Cisco、OpenWrt 等。

⌨

终端宏录制

录制登录和命令序列，一键回放。ANSI 过滤确保宏干净无干扰，延迟自动压缩至即时响应。

💾

SFTP 文件管理

基于 SSH 的完整 SFTP 客户端。浏览、上传、下载、创建目录、查看元数据、删除 全部可通过 MCP 工具或内置面板操作。

🌎

中英双语

完整的中文（zh-CN）和英文（en-US）本地化。菜单、对话框、工具描述 全部双语覆盖。

🛡

默认安全

MCP 服务仅绑定 localhost、Bearer Token 认证、CORS 来源限制。你的设备始终受到保护。

📈

模式监控告警

支持正则表达式的模式监控。匹配 panic/watchdog/error 等关键模式，实时推送 MCP 通知，结果自动持久化。

MCP 工具集

42 个 AI 就绪的工具

每个连接、数据和会话操作都暴露为结构化 MCP 工具，AI 客户端可自动发现并调用。

connconn.create创建串口/SSH/Telnet/PTY 连接
connconn.data.write发送文本，支持延迟和条件等待
connconn.data.write_hex发送十六进制帧（Modbus 等）
connconn.data.expect等待输出中出现指定模式
scriptconn.script.login自动登录，未知提示符触发 AI 决策
scriptconn.script.run执行多步骤测试脚本
watchconn.watch.start模式监控 + 实时告警通知
watchconn.watch.results查询持久化的告警历史
recordconn.record.start终端会话录制
recordconn.record.replay1x / 2x / 即时速度回放
analyzeconn.analyze.probe21 种设备自动探测
analyzeconn.analyze.report会话统计摘要报告
hwconn.hw.dtr_rtsDTR/RTS 信号控制
hwconn.hw.break串口 Break 信号
fileconn.file.sendXMODEM/YMODEM 文件传输
sftpsftp.connect通过 SSH 建立 SFTP 会话
sftpsftp.upload上传文件到远程主机
sftpsftp.download从远程主机下载文件
sessionsession.save保存连接配置为会话
devicedevice.ports列出可用串口列表
appapp.screenshot捕获终端窗口截图
appapp.macro.run回放已保存的宏
connconn.shareTCP 连接共享
