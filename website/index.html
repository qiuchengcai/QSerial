Features

Everything an embedded engineer needs

From serial debugging to AI-powered automation, QSerial covers the full embedded development workflow.

🤖

MCP AI Integration

42 built-in MCP tools let AI assistants control your devices. Resources, Notifications, Sampling, and Prompts all standard.

🔌

Multi-Protocol

Serial (COM/tty), SSH (with Jump Host), Telnet, and local PTY. All in one unified interface with consistent tooling.

🔍

Auto Device Detection

conn.analyze.probe identifies 21 device types: ESP32, STM32, NXP i.MX, Raspberry Pi, Cisco, OpenWrt, and more.

⌨

Terminal Macro Recording

Record your login and command sequences. Replay with one click. ANSI filtering keeps macros clean of control characters.

💾

SFTP File Management

Full SFTP client over SSH. Browse, upload, download, mkdir, stat, and delete all via MCP tools or the built-in panel.

🌎

i18n Ready

Full Chinese (zh-CN) and English (en-US) localization. Menu, dialogs, tool descriptions everything is bilingual.

🛡

Secure by Default

MCP server binds to localhost only. Bearer token auth. CORS origins restricted. Your devices stay safe.

📈

Watch & Alert

Pattern monitoring with regex support. Match panic/watchdog/error patterns and get real-time MCP notifications.
